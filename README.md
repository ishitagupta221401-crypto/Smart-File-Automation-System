# Smart File Automation System

## Overview

This project was developed as part of the Alfido Tech Internship Program.

The Smart File Automation System automates common file management tasks such as file creation, reading, writing, renaming, moving, deletion, CSV processing, and activity logging.

## Features

* Create File
* Read File
* Write File
* Rename File
* Move File
* Delete File
* Read CSV File
* Activity Logging
* Exception Handling

## Technologies Used

* Python
* OS Module
* SHUTIL Module
* CSV Module
* Datetime Module

## Project Structure

* src
* data
* logs
* output
* screenshots
* docs

## How to Run

1. Open terminal
2. Navigate to src folder
3. Run:

python main.py
